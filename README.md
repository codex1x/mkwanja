# Mkwanja
Peer to peer money sharing app

# Figma Link
https://www.figma.com/file/XnrjekAsNtWJrygYxs6LGb/MKWANJA-APP?node-id=0%3A1

# App
The app is flutter and is found in mobileapp folder. Compile and run.

# Backend

The backend is found in mkwanja-base-api folder. The app is Spring Framework based. I have tried to show the proper code structure for such project.
It is docker enabled but can be run normally in IDE. There is a script called start.sh for quick start in docker but also a normal docker command to run a docker file may be used.
SMS Gateway is integtrated for sending OTP.
